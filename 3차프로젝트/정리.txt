1. 임대진
대신투자증권 : 
1. 차트가 너무 많아서 초보자가 차트를 파악하기 어려웠다. 
2. 밑의 메뉴바를 선택하면 초기화되어서 불편 
3. 버퍼링이 너무 심하다.
4. 메뉴가 너무 많아서 뭐부터 손대야 할지 모르겠다.

유진투자증권 : 
1.  계좌 계설용 어플을 하나 더 설치해야함, 
2.  주민등록증 사진 찍는게 신청을 기다려야함

희망 기능 : 
1. 주식 용어들을 직관적으로 알기 쉽게
2. 매수 매도를 바로 직관적으로 바꾸기
3. 차트를 간소화 

2. 서민규
희망기능 : 
1. 메뉴 차트를 길게 누르면 그 용어에 대한 정보를 확인할수 있으면 좋겠다,(롱클릭) 
2. PC에서 사용했던 관심정보가 있으면 바로 가져올 수 있도록
3. 그래프 보는게 불편하기에, 차트 확인할 때는 가로기능이 있었으면 한다.
4. 상단바, 하단바로 메뉴가 나뉘어져 있는데 이걸 모두 상단바로 올렸으면 좋겠다.

3. 강다현
토스 : 
1. 관심주, 소유주 중심이라 실시간 거래가 많은 주식은 알 수 없음 
2. 등락률 현재가 등 실시간으로 변하는 여러 주의 지표를 한눈에 보지 못함
3. 종목의 현재가를 실시간으로 확인하기 어렵다. 어떤 지표 순위로 정리되어 있는 정보가 없다.
4. 종목 종목의 정보를 확인하는건 토스가 좋다.

나무 : 
1. 주식외에도ETF등 다른 것들이 많아 한눈에 들어오지 않음, 
2. 창 안에 글씨가 다 안들어와서 불편하다,
3. 가로스크롤이 확인하기에 불편

4.윤정은
1. 종목간 비교차트 기능이 있으면 좋다.
2. 저널 사이트와 연계해서 어려운 개념 등을 해설해 주는 뉴스레터를 제공
3. 서치할수 있는 창이 메인에 있으면 좋겠다.
4. 메뉴를 간소화시켜서 확인할 수 있게 
5. 여러가지 종목 추천 서비스를 제공함

5. 정혜진
오늘의 투자 : 
1. 가입이 간단함
2. 투자성향 설문조사를 함 이에 맞춰 종목추천을 해줌(꽤나 유용해보임)
3. 요약표 정보만 제공(시가 종가 고가 저가 등등 단편적)
4. 용어 설명이 쉬움, 
5. 그래프가 너무 간단
6. 금융상품 추천이 많다.(불편함)
7. 캔들차트를 선택할 수 있게,

삼성증권:
1. 가입이 6단계 인증, 너무 복잡
2.  종목별 개인, 외국인, 기간투자자, 기타 투자자 거래량 확인
3. 1분차트 - 1분 간격으로 짧은 간격으로 확인가능
4. 토론 페이지 :  다음 네이버 댓글 토론 페이지

6. 정선일
1. 시작페이지 설정 기능을 주면 좋겠다. 커스터마이징 할수 있게
2. 내가 이 주식을 열흘전에 샀다면 지금 수익률이 몇%인지, 지금 사면 열흘 후에 수익률이 몇%일지 예측해주면 좋겠다.
3. 어플들 간의 정보 제공량 격차가 너무 크다, 사용자에게 적합한 선으로 맞춰야함
4. 초보자가 자주 쓰는 거래방식과 정보만 확인 가능하도록 메인페이지를 구성해야함
5. 차트를 나열해주는 기준을 사용자에게 제시해 줘야함
